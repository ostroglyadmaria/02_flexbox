HTML

HTML (HyperText Markup Language, мова гіпертекстової розмітки) — стандартна мова для створення структури вебсторінки. HTML дозволяє визначити заголовки, абзаци, списки, таблиці, зображення, форми та інші елементи інтерфейсу. HTML-документи складаються з елементів, які позначаються тегами, наприклад , тощо. Основні особливості: Визначає логічну структуру контенту Підтримує вбудовані мультимедійні елементи (аудіо, відео) Використовується у поєднанні з CSS і JavaScripеt

CCS

CSS (Cascading Style Sheets, каскадні таблиці стилів) — мова, яка використовується для опису зовнішнього вигляду HTML-елементів. CSS дозволяє змінювати кольори, шрифти, відступи, розміщення елементів на сторінці, додавати анімації та адаптивність до різних екранів. Рівні CSS: CSS1 — початкова специфікація CSS2 — додано макет, таблиці, позиціонування CSS3 — розділено на модулі, додано нові можливості (тіні, анімації, медіа-запити)

JS

JavaScript (англ. / ˈ dʒ ɑː v ə s k r ɪ p t / ; аббр. JS ) - мультипарадигменна мова програмування . Підтримує об'єктно-орієнтований , імперативний та функціональний стилі. Є реалізацією специфікації ECMAScript (стандарт ECMA-262 [ 2 ] ). JavaScript зазвичай використовується як вбудована мова для програмного доступу до об'єктів додатків . Найбільш широке застосування знаходить у браузерах як мову сценаріїв для надання інтерактивності веб-сторінок [ 8 ] . Основні архітектурні риси: динамічна типізація , слабка типізація , автоматичне управління пам'яттю , прототипне програмування , функції як об'єкти першого класу . На JavaScript вплинули багато мов, при розробці була мета зробити мову схожою на Java . Мовою JavaScript не володіє будь-яка організація або компанія, що відрізняє його від низки мов програмування , що використовуються у веб-розробці [ ~1 ] [ 9 ] . Назва "JavaScript" є зареєстрованим товарним знаком корпорації Oracle в США [ 10 ] .
